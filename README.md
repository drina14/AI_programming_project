   Student grade predictor AI chatbot
   
   ## Description
   The Student Grade Predictor Chatbot is an intelligent conversational AI system that helps students predict their final grades and understand their academic performance.      The chatbot combines rule-based conversation with grade calculation logic to provide personalized academic insights.
   - Interactive Conversation: Students can chat naturally with the bot about their grades
   - Grade Prediction: Calculates predicted final grades based on CAT scores, assignments, and attendance
   - Study Recommendations: Suggests focus areas based on performance patterns
   
   ## Libraries Used
   **Python 3.8+**: Core programming language
- **Built-in Libraries Only**: 
  - `sys` - System operations
  - `datetime` - Timestamp functionality (optional)
  - No external dependencies required
   
   ## How to Run
   1. Clone this repository:
```bash
   git clone https://github.com/drina14/AI_programming_project
   cd AI_Programming_Project
```

2. Run the chatbot:
```bash
   python main.py
```

3. Interact with the bot by typing your questions

4. Type 'exit' or 'quit' to end the session
